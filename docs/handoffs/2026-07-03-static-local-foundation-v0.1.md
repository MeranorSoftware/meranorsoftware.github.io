# Meranor Website - Static Local Foundation v0.1

## Summary

- Replaced the Astro and Node-based architecture with a pure static local-first website.
- Preserved the approved Meranor copy, product boundaries, and workshop sample content.
- Added shared static styling, a replaceable local SVG maker's mark, and real local workshop pages.

## How To Open

1. Open [index.html](/C:/Users/jwelch/meranor-site/index.html) directly in a browser.
2. Optional: if Python is installed, use `python -m http.server 8000` for a local preview server.

## Notes

- No Node or npm is required.
- No build step exists.
- All current links are local and relative.
- The workshop pages remain sample content while the long-term publishing workflow is still being defined.
