## How They Should Be Used

- `meranor-homepage-north-star-d.png` is the canonical homepage composition reference.
- `meranor-brand-card-v1.png` is the canonical brand reference for the maker's mark, palette, spacing, and tone.
- `meranor-promo-poster.png` is cinematic mood reference only.

## What The Static Site Should Keep

- a wide cinematic hero with a horizon glow
- calm premium charcoal surfaces
- warm brass emphasis instead of generic tech blue
- clear spacing rhythm and rounded panels
- a software-first feel that stays restrained and intentional

## What The Static Site Should Avoid

- cluttered visual treatments
- neon gradients
- heavy glassmorphism
- cyberpunk styling
- anything that makes Meranor look like a game studio or crypto brand

## v0.4 Asset System Note

The homepage now uses dedicated local SVG assets for product marks, the FieldKeeper product preview, and workshop editorial thumbnails. These are not external dependencies and are intended as first-pass brand assets that can be replaced as each product receives final iconography or real screenshots.
