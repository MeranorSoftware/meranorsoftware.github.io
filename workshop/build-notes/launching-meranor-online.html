Build Notes

Launching Meranor online

The first public Meranor website is live: a clean static home for the company, the product ecosystem, and the workshop notes that will document what we build next.

Public release July 3, 2026

Release focus

This Build Note summarizes the first GitHub Pages launch and turns the internal changelog into public context without exposing the messy details that belong inside the workshop.

Meranor now has a public home. The first release is intentionally simple: static HTML, CSS, and a small amount of vanilla JavaScript hosted through GitHub Pages.

The goal of this launch was not to pretend every product is finished. The goal was to create a clear, honest place for the company vision to live while FieldKeeper, Astra AI, Geomancer, PatchCube, and CountLens continue to develop.

What changed in this launch

  • Established the Meranor brand, mark, tagline, and public mission.
  • Published the first product ecosystem overview.
  • Added a founder page and workshop structure.
  • Replaced temporary visual drafts with the approved brand mark and product imagery.
  • Launched the site on GitHub Pages for a simple public foundation.

Why Build Notes exist

A private changelog is useful for development. A public build note has a different job: it explains progress in a way that is understandable, honest, and useful to people following the work.

That means future changelog updates can become short public notes when they matter: major design locks, product milestones, FieldKeeper progress, Astra AI experiments, Geomancer releases, or changes to the Meranor site itself.

What comes next

The next step is to keep improving the public vision without over-claiming. The Workshop will carry product thinking and build updates. The homepage will stay focused on clarity. The product cards will remain honest about what is live, in development, or still in concept.

Meranor is now online. The work is still early, but the direction is public: build thoughtful software, build it carefully, and build what matters.
